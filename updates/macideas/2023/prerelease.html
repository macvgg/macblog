MacIdeas Prerelease Update Log

8.2.2023 ~ 10.23.2023

MacIdeas, this amazing productivity app by MacWeb, has been through A LOT of updates, changes, and revamps. It was originally developed and called Ify in the August of 2023, and then got acquired by MacWeb in fall 2023 and was renamed to MacWork with a brand new interface, appearance, features, and many others new features planned to be added but wasn't published. It got abandoned for a while but picked up again in April 2024, and was completely reimagined again. It was renamed to MacIdeas and had a new roadmap, new appreance, new features, and more planned in the future. See all the updates from these seperate phases below! For all the updates after the June 2024 release of MacIdeas, see the detailed dedicated articles for them on the left sidebar! The prerelase logs aren't detailed and it's very chaotic, so beware.

v.0.5 | 10/23/2023

Ify acquiered by MacWeb and renamed to MacWork! Ify was officially merged into MacWeb as MacWork, one of its apps! We are very excited to see more features, changes, improvements, and updates to come! Goodbye Ify, hello MacWork, the most powerful productivity app! All the features, appearance, and all the pages were entirely transformed into the new MacWork and the app organization was completely changed. We moved many pages around and deleted some unneccessary files and folders. Now you can have the easiest navigation between apps! We will still publish the complete and powerful versions of MacWork as an app and an extension like previously promised to boost your productivity even more! New apps: Docs and MacText! What's a productivity app without a text editor? Or two text editors?! We are proud to announce that we're adding Docs for powerful text edits and MacText for powerful text modification! Docs can do anything a text editor can, and beyond! You can type paragraphs, format them into nice-looking articles, print them, add imges and media, put special effects, and more all on the app! MacText helps your paragraph become better! It can correct your grammar, shorten text, capatilize all, and more! Overall performance improvements and updates. New app appearance! With the acquierance of Ify, the app is also getting a complete visual revamp and user interface! We added more MacWeb elements so your experience would keep getting better and better! We removed the Ify sidebar and added the horizontal navigation bar on the top of the page for quicker access between each MacWork apps for direct integration and easier UI and UX. We restyled all the elements and apps to match the overall MacWeb theme and repositioned all the elements for better UX! Huge performance improvements, updates, and a lot of bug fixes.

v.0.2 | 9/10/2023

New Feature: Notes! What's the most productive way to create an idea, write it down, and share it to others? That's right, Ify Notes! Whenever you're learning something or having a new idea, keep it in Ify Notes! You can now add notes, edit them, and save them from an easy to use user interface! New user interface: edit notes directly in the textarea at the center of the page, save them using the Save Notes button at the bottom of the page, and brand new page layout. New Page: Ify Support! Don't know how to use a feature in Ify? No problem, Ify Support's got you covered! You can find each feature's detailed descriptions and easily navigate through them! Don't forget to drop a feedback or ask any questions using Ify Feedback! Improved User Interface and Functionalities Updated user interface Appearance and user special effects. Bug fixes, including task editing not working, tasks saving, notes saving not working, and other functionality improvements. Overall performance improvements and minor updates.

v.0.1.6 | 8/26/2023

New Feature: Tasks Editing! Want to edit your task for a better idea? No problem, Ify Tasks Editing has you covered! Just click the pen icon on the tool bar and start editing your text! You can add or remove text, add emojis, and any symbol to your task item! After finished editing the task, click anywhere else on the screen and the changes will be automatically saved! New Feature: Tasks Drag & Drop! In order to boost your productivity even more, you can now reorder tasks for your works and priority tasks! Simply drag the task item to where you want it to be in the task list and drop it on the blue line indicating the position! Improved User Interface and Functionalities Fixed task managing interface and new task completing effect. Appearance and effects improvements. Improved sidebar navigation. Bug fixes, including remove button animation, tasks dragging bug, tasks editing not working, and other functionality improvements. Overall performance improvements and minor updates.

v.0.1.3 | 8/2/2023

Introducing Ify: Let Your Ideas Fly! Ify: Let Your Ideas Fly is an app that boosts your ideas and productivity! Ify is the collection of 4 must-needed productivity apps for your work! Set your goals using Ify Tasks; write and draw your ideas down using Ify Notes and Ify Sketches; and create focusing sessions on your work using Ify Focus! Even better, it's completely free and ad-free, with web version and browser extension version so you can get help from Ify anywhere, anytime! Ify beta version officially launched on GitHub for testing! New Feature: Tasks! Tasks made easy. Add your works and goals to Ify Tasks and boost your productivity to finish them all! You can now add tasks, complete them, and remove them from an organized tasks list! New user interface: input bar at the bottom of the page, create task button in the input bar, new task list item style, and brand new elements placement. New Page: Updates! See what's new with Ify in the Updates page and try them out! Keep track of Ify updates and know all the changes! New User Interface and Functionalities Brand new Ify interface with dark appearance. Sidebar for easier navigation between Tasks, Notes, Sketches, and Focus. New logo design and new links. More comfortable section and link designs. Bug fixes, including tasks not removing from localStorage, no ding sound effect when completed task, and scripting connections. Overall performance improvements and minor updates.

Other Improvements, Changes, and Fixes include:
  • Renamed the app from Ify to MacWork, and finally to MacIdeas on release with many major changes on the basic functionalities of the app and its purpose.
